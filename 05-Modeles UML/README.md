# Modèles UML — FOOSUS

Ce dossier rassemble les diagrammes UML locaux produits pour illustrer les interactions, les composants et les structures internes de la plateforme FOOSUS.  
Ces modèles complètent les vues d’architecture en apportant un niveau de détail opérationnel.

## Contenu du dossier

- Diagrammes de séquence  
- Diagrammes de classes  
- Diagrammes de composants  

## Rôle dans l’architecture FOOSUS

Les modèles UML permettent de préciser les comportements, les responsabilités et les échanges entre services.  
Ils servent de support aux discussions techniques et facilitent la compréhension des mécanismes internes.
